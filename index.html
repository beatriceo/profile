Projects

Seam Carving

A program that implements seam carving, a content-aware image resizing algorithm developed by Shai Avidan and Ariel Shamir. Important details of images are preserved while "low-energy" pixels are removed. The energy of each pixel is calculated by finding the rgb difference between adjacent pixels. A seam containing the lowest energy pixels are removed. The pixel energy is then recalculated following each seam removal. This processed is repeated n-times for n-pixels to be removed from the original image.

For more information, view our project report here.

GIFLESS

A program where users could search for a film quote in a database of films and can generate a GIF containing the scene and subtitles from their query. This group project was the first place sub-honours winner in the "Hack the Bubble" Hackathon at the University of St Andrews in December 2017.
